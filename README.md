# Zombie Game
- 자바로 다형성 콘솔 게임을 구현하였습니다.
	- 좀비 사냥
	- 유령 사냥
	- 마법사 사냥
	- 보스 사냥

#### 게임 설명
- 일직선 던전을 이동하며 몬스터들을 만나서
- 랜덤 데미지 공격으로 전투를 하며 몬스터들을 처치하고
- 마지막까지 살아남아 보스까지 처치하면 클리어하는 게임
